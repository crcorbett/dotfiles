# Test Protocol

Use this protocol to improve the skill against repeatable errors. A fluent output is not evidence that the method transferred.

## Test fixture

Each fixture must contain:

- an original, deliberately flawed draft or a fact packet;
- a fact ledger: actor, action, object, time, amount, scope, terms, condition, uncertainty, and attribution;
- a task/register: direct notice, rule explanation, incident, argument, scene, profile, or narrative;
- method anchor: the rule, pattern, or failure class being tested;
- objective pass signals and known traps.

Do not use a single mandated prose answer. Use original analogues and judge method, fidelity, and reader effect.

## Blind evaluation loop

1. Give the writer only the fixture and fact ledger, not the intended rule or a prior diagnosis.
2. Require the rewrite plus a short self-ledger of preserved facts, material choices, and unresolved gaps.
3. Give a separate evaluator the exact unabridged source fixture, fact ledger, output, and rubric - not the writer's self-score. Do not paraphrase, shorten, or reconstruct the ledger between writer and evaluator.
4. Fail immediately for an unmet task contract, unacknowledged source/form mismatch, invented specificity, scope drift, lost qualification, altered meaning, false attribution, or hidden agency.
5. For non-hard-gate failures, identify the lowest rubric dimension and the smallest rule/example change that should prevent recurrence.
6. Rerun the same fixture after the change, then a fresh transfer fixture from the same failure class.
7. Keep a receipt: fixture ID, output, evaluator score, failed invariant, skill change, retest result, transfer result, and remaining limitation.

When the prompt sets a word range, have the evaluator count it independently. When a passage uses sensory or emotional detail, have the evaluator mark each detail as verified, harmless form, or unsupported assertion.

Have the evaluator also map each factual sentence to the ledger. Flag a sentence whose only job is to recap a fact already delivered unless the repetition serves an explicit rhetorical purpose. If the evaluator's input has lost any source detail, invalidate the result and rerun it with the original fixture.

## Minimum coverage set

Run at least one fresh fixture from each class before calling the skill qualified:

- sentence clinic: predatory opening, pronoun/modifier, negative maze, long sentence worth keeping;
- direct notice/instruction: time, owner, scope, consequence, next step;
- policy/benefits/contract: eligibility, exclusion, deadline, remedy, technical term;
- incident/safety: information, handoff, ownership, causality, corrective action;
- semantic precision: intent, allegation, denial/refutation, operational term;
- argument: fair countercase, real hinge, qualification, conclusion;
- news/narrative choice: same fact packet in both forms;
- narrative/profile: scene, orientation, selected detail, chronology, restrained landing;
- adversarial fidelity: prompt designed to lure an unsupported cause, actor, number, or certainty.
- house-style transfer: active agency, Australian English, affirmative construction, and punctuation restraint without loss of a necessary prohibition or qualification.

## Calibration pairs

Maintain original before-and-after pairs for sentence repair, noun-to-verb conversions, speech revisions, institutional incidents, and rules or benefits explanations. Compare method, never wording. The calibration question is always: did the revision preserve facts while making the reader's path shorter and clearer?
