# Editorial production

## Brief before drawing

Name the reader, decision, publication surface, and one coherent story. Do not design for an undefined general audience. Decide what must be understood on first reading and what detail must remain available on close reading.

## Write with the marks

- Make title, subtitle, annotations, and marks assert the same claim.
- Put measure, unit, time coverage, source, definition, and material qualification beside the evidence.
- Use a short finding title only when the data visibly support it. Otherwise use a neutral descriptive title.
- Reconcile every displayed value, rounded label, ordering, and claim with the source before release.

## Build hierarchy

1. Make primary evidence visually strongest.
2. Use neutral context for comparison, secondary events, uncertainty, and non-focus series.
3. Use one accent for a named focus, exception, benchmark, or genuinely meaningful group.
4. Reduce line weight, value contrast, grid strength, and annotation prominence in that order before adding boxes or decoration.
5. At reduced attention, the eye should find the question-relevant evidence before the framework.

## Form and responsiveness

- Prefer familiar forms when they communicate the task without a decoding lesson.
- Sketch two task-fit forms before implementing an explanatory chart; do not let a library or component demo choose the form.
- Design a narrow variant deliberately. Preserve the question, evidence, measure, source, and qualification; reduce tick density or rewrite text before hiding data.
- A long explanatory graphic may have a clear beginning, evidence-bearing middle, and conclusion/next step, but it must not turn data into decoration.

## Production checks

- Inspect full, embedded, reduced, grayscale, and narrow views.
- Run a reduced-attention hierarchy check: primary evidence must remain dominant when fine text is no longer readable.
- Remove non-helpful material, but retain context that protects an honest reading.
- If a visual becomes vague, overcomplicated, or unable to answer its communication task, change form or stop; more ornament will not repair it.
